Problem 1: 2-to-1 1-bit mux

For this problem you will implement a 2-to-1 1-bit mux. Given input A,B, and S; if S is low then A is assigned to Y, if S is high then B is assigned to Y. By assigned, I mean whatever value the former takes will be shown for the latter.

So if A = 1, B=0, S=1, then Y =1

module main(

input A,

input B,

input S,

output Y);

endmodule
